# MonksTrip Travel Agency

Welcome to the MonksTrip Travel Agency static template.

A luxury, dark-mode supported destination and travel booking prototype website.

## Directory Structure
- `assets/` - Images and public graphics
- `data/` - JSON files like blogs data
- `lib/` - JavaScript modules, including Supabase client
- `pages/` - Individual HTML pages for destinations, services, blog, etc
- `index.html` - Main landing page

## Deployment
This template is optimized and pre-configured for instant deployment on Vercel. 
The static site will resolve properly using the root `index.html` and `vercel.json` rewrite configuration.
